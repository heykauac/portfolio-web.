Meu Primeiro Projeto Acadêmico

Descrição: Este projeto foi desenvolvido como artefato do 1º período. Trata-se da estruturação de um sistema utilizando lógica de programação e conceitos básicos de desenvolvimento.

Desafios: O maior desafio foi aplicar os conceitos de CSS Grid e Flexbox para manter a interface limpa e responsiva, além de estruturar o código de forma semântica.
