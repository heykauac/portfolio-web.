# Portfólio de Kauã

Site de portfólio desenvolvido com HTML, CSS e JavaScript para o 1º período.

## 🛠️ Tecnologias e Critérios 
- **Layout:** Estruturado utilizando CSS Flexbox (Navegação) e CSS Grid (Container principal).
- **Semântica:** Uso de tags como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` e `<footer>`.
- **Navegação:** 3 páginas interligadas (`index.html`, `projetos.html`, `estudos.html`).
- **JavaScript:** Funcionalidade de Dark/Light mode e atualização de ano no rodapé.
- **Validação:** Código testado e validado no Markup Validation Service (W3C).
